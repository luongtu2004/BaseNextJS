🔑 Manual Token Input

Bước 1: Login bằng form OAuth2 (click 🔒 → Authorize → nhập phone/password → Authorize)

Bước 2: Copy access_token từ response → Paste vào ô dưới → Click "Set Token"

📖 Xem hướng dẫn chi tiết

1. Token Login:

{
  "phone": "0912345678",
  "password": "your_password"
}

2. Token Response:

{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": { ... },
  "roles": ["admin"]
}

📖 Quick Authentication

  • Username = Phone: Nhập số điện thoại vào ô username
  • Không cần client_id/secret: OAuth2 Password Grant không yêu cầu
  • Token hết hạn sau: Xem config jwt_expire_hours
